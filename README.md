# forget-me-not
a slow letter platform for people who believe real things take time.

## what is this?

forget-me-not is a web platform where people write letters to each other - and those letters actually travel based on real geographic distance. istanbul to tokyo? that letter takes days. and those days of waiting are the whole point.

- real letter delivery system with distance-based travel time
- ai compatibility matching and ice-breaker suggestions
- bottle mail — throw a letter into the ocean for a stranger to find
- memory box — private letters you write but never send

no instant messaging. no read receipts. no "wyd" at 2am. just real letters, real patience, real connection.

## why i built this

the internet was supposed to connect us, but it made us lonelier instead. love is measured in response times. human connection became fast food — quick, convenient, forgettable.

i wanted to build something for people who still believe that slow things are the most beautiful things. a place where every letter feels like unwrapping a gift, and every word carries weight.

this is for the dreamers, the overthinkers, the ones who would have written letters in another century.

## tech stack

- **frontend:** Next.js (App Router) + custom CSS design system
- **backend/db:** Supabase (PostgreSQL + Auth)
- **auth:** Google OAuth
- **hosting:** Vercel (planned)
- **payments:** Stripe (planned)
